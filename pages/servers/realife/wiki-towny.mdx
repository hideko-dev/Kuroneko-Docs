# Realife Wiki

`Towny`<br/>
**/t new 町の名前** ←新しく町を作成します。多分800円で作れ、1日10円の維持費がかかります。<br/>
**/t spawn** ←町にテレポートします。<br/>
**/t add mcid** ←町に入れたい方に招待を送ります。/acceptで承認されれば町民になります。<br/>
**/t claim** ←町の領土を増やします。領土には所得上限があります。<br/>
**/t claim outpost** ←町に飛び地を作ります。作りすぎると重くなるので注意<br/>
**/t spawn outpost** 値 ←作ったoutpostにテレポートします。<br/>
**/t toggle mobs on/off** ←町の領土内に敵モブが湧くか湧かないかを切り替えます。<br/>
**/t toggle pvp on/off** ←町の領土内でpvpができるかを切り替えます。<br/>
**/t toggle fire on/off** ←町の領土内で火が燃焼するかを切り替えます。<br/>
**/t toggle public on/off** ←町民以外が町にテレポートできるかを切り替えます。<br/>
**/t set spawn** ←/t spawnでテレポートするスポーン地点を変更します。<br/>
**/t set title mcid 肩書き** ←町民に肩書きを付けます。色を付けたい場合は&1~9などのカラーコードをご使用ください。<br/>
**/t set perm on/off** ←町の全権限を切り替えます。村長以外何もできなくなります。<br/>
**/t set perm resident on/off** ←町民の全権限を切り替えます。<br/>
**/t set perm outsider on/off** ←町に入ってない方の全権限を切り替えます。荒らせてしまうので使用は推奨しません。<br/>
**/t deposit 値** ←町の銀行にお金を入れます。<br/>
**/t withdraw 値** ←町の銀行からお金を引き出します。<br/>


`Plot編 (ここで表現されるplotとは現在いる1チャンクだけが対象となります。)`<br/>
**/plot toggle mobs on/off** ←plotの領土内に敵モブが湧くか湧かないかを切り替えます。<br/>
**/plot toggle pvp on/off** ←plotの領土内でpvpができるかを切り替えます。<br/>
**/plot toggle fire on/off** ←plotの領土内で火が燃焼するかを切り替えます。<br/>
**/plot set perm on/off** ←plotの全権限を切り替えます。村長以外何もできなくなります。<br/>
**/plot set perm resident on/off** ←plotでの町民の全権限を切り替えます。<br/>
**/plot set perm outsider on/off** ←町に入ってない方の全権限を切り替えます。荒らせてしまうので使用は推奨しません。<br/>
**/t rank add  mcid assistant** ←他の方を町に招待する権限を渡します<br/>
